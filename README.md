# Vokabellernen

## Hello 👋 Welcome to my first project using QT and C++.

### What is the aim of this project :question:
First of all, a bit of history about my learning of English.

When I learn English vocabulary I write the word in English and in French on the same line in an EXCEL table. Then I print and cut each line. I fold them and put the pieces of paper in a plastic pocket and I draw lots to guess either the word in French or in English. 

So I wanted to do the same with an application on my computer to learn german words. 

My application is connected to a database. One page of my application allows me to add a word to my database, another to draw a word and choose if I want to guess the French or German translation, another page allows me to guess the article of a noun (das der or die) and another to guess the plural form.


### But Agathe, this type of application already exists, so why do you want to rebuild another one?

Yes, I know. But I discovered the field of software and especially UI creation in my current company. I really like it!!! I really like the fact that I mix the "artistic" field with programming. So I decided to create my own software and to learn a new language: C++.
